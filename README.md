# Student-performance-analyzer-

A full-stack web application designed to track and visualize student academic performance.

## 🚀 Features
- **Data Entry:** Input student names and marks for multiple subjects.
- **Automated Logic:** Backend calculates Total, Average, and Grade.
- **Visualization:** Dynamic bar charts using Chart.js.
- **Persistent Storage:** Data is saved securely in a MySQL database.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6), Chart.js
- **Backend:** Node.js, Express.js
- **Database:** MySQL (XAMPP)

## ⚙️ How to Setup
1. Clone this repository.
2. Open XAMPP and start **Apache** and **MySQL**.
3. Create a database named `student_db` and run the SQL provided in the docs.
4. Run `npm install` to install dependencies.
5. Start the server using `node server.js`.
